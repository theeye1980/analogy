Внедрение на свой сайт за 2 часа - 4 простых шага!

1 Шаг - зарегистрироваться в личном кабинете analogi.pro

2 Шаг - Добавить ваши товары в сервис в формате Yandex YML

Формат Яндекс YML

Если вы продаете на маркетплейсах, то скорее всего, у вас требуемый фид уже есть. В случае, если нет, то обычно программист, который занимается поддержкой сайта может создать для вас за 0,5-1 рабочий час времени в зависимости от системы управления контентом вашего сайта.

Для регистрации достаточно указать свой телефон, имя и электронную почту. Телефон и почта понадобится для получения одноразового пароля.

3 Шаг - Дождаться создания кропов (индекса картинок товаров) - 6..20 минут

Чтобы осуществить поиск по изображениям ваших товаров, наша система анализирует все картинки и создает кропы (специальные индексы товаров), по которым далее будет осуществляться поиск. Среднее время обработки 5000 товаров - около 10 минут. Система в процессе обработки ваших товаров сообщает о текущем статусе.

4 Шаг - Выбрать тип интеграции

Имеется возможность внедрить на сайт двумя способами:


1. Использовать предлагаемые нами виджеты


2. Осуществить внедрение с помощью предлагаемой API (ссылка на документацию для разработчиков здесь)

Внедрение с помощью виджета подразумевает следующее:


Вы настраиваете внешний вид отображаемых товаров в личном кабинете и получаете js код, который выводит перечень товаров, похожие на тот, что вы передаете в специальный тег на странице. А сам код устанавливаете в то место, где хотите получить список товаров.


Этот тип интеграции способен выполнить Вебмастер и контент менеджер, не являющийся программистом
